Problem Title: Graph Connectivity Queries (BFS)
Contest: Round 1: Shattered Syntax (Debugging)
Problem Type: Debugging
Problem Mode: Standard Input/Output
Total Marks: 30

Constraints:
1 <= N <= 2e5; 0 <= M <= 2e5; 1 <= Q <= 2e5

Description:
Given an undirected graph and Q queries, determine for each query if there exists a path between u and v.

Input Format:
N M Q
(M lines) u v
(Q lines) u v


Output Format:
For each query: Yes or No
