Problem Title: Array Rotation
Contest: Round 1: Shattered Syntax (Debugging)
Problem Type: Debugging
Problem Mode: Standard Input/Output
Total Marks: 10

Constraints:
1 <= N <= 2e5; 0 <= K <= 1e9; |ai| <= 1e9

Description:
Rotate an array of N integers to the right by K positions and print the rotated array.

Input Format:
N K
a1 a2 ... aN


Output Format:
Rotated array elements space-separated
