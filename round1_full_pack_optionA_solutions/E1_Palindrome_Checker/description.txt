Problem Title: Palindrome Checker
Contest: Round 1: Shattered Syntax (Debugging)
Problem Type: Debugging
Problem Mode: Standard Input/Output
Total Marks: 10

Constraints:
1 <= T <= 50; 1 <= |s| <= 1e5 (sum lengths <= 2e5)

Description:
Given T strings, print Yes if the string is a palindrome else No. Case-sensitive; use the full string as-is.

Input Format:
T
s1
s2
...


Output Format:
For each string: Yes or No
