Problem Title: Merge Intervals
Contest: Round 1: Shattered Syntax (Debugging)
Problem Type: Debugging

Constraints:
1 <= N <= 2e5; -1e9 <= start,end <= 1e9; start <= end

Description:

Input Format:
N
start1 end1
...


Output Format:
Merged intervals, one per line: start end
