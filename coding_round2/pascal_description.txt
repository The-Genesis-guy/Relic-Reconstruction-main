Problem Title: Pascal Triangle
Contest: The Architect's Blueprint - Round 2
Problem Type: Coding
Problem Mode: Function
Total Marks: 10

Constraints:
1 <= n <= 30

Description:
Given an integer n, generate the first n rows of Pascal's triangle. Each row's values are separated by a single space.

Input Format:
Single integer n

Output Format:
Print n lines; line i (1-indexed) contains the i-th row of Pascal's triangle with values space-separated.
