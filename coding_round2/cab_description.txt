Problem Title: Safe Night Cab
Contest: The Architect's Blueprint - Round 2
Problem Type: Coding
Problem Mode: Function
Total Marks: 30

Constraints:
1 <= N <= 2e5
0 <= P[i] <= D <= 1e9
0 <= R <= 1e9

Description:
The city is a straight road from position 0 to position D. There are N street lights. The i-th light is at position P[i] and illuminates the segment [P[i]-R, P[i]+R].
The cab can travel only through illuminated segments and can start/stop only within illuminated areas. Determine if the cab can safely travel from 0 to D.

Input Format:
First line: N D R
Second line: N space-separated integers P[i]

Output Format:
Print YES if travel is possible, otherwise NO.
