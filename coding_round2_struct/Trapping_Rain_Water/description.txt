Problem Title: Trapping Rain Water
Contest: The Architect's Blueprint - Round 2
Problem Type: Debugging
Problem Mode: Standard Input/Output
Total Marks: 40

Constraints:
1 <= n <= 2e5
0 <= height[i] <= 1e9

Description:
Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it can trap after raining.

Input Format:
First line: integer n
Second line: n space-separated integers height[i]

Output Format:
Single integer: total trapped water.
